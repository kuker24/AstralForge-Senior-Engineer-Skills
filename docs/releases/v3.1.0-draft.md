## Summary

v3.1.0 is planned as an evidence-hardening and Pi core-tools release for AstralForge Senior Engineer Skills. It preserves the `v3.0.0` release tag at commit `17ea37e` and treats the final post-release main baseline `77f2353` as separate evidence cleanup. The release adds Context7 CLI + Skills guidance, Serena availability, Repomix token reporting, token profiles, local Pi apply/verify/cleanup scripts, and the Skill Security Auditor without claiming external certification.

## Local Validation Required Before Publishing

Minimum local checks:

```bash
npm run typecheck
npm run test:unit
npm run test:coverage
npm run verify:skills
npm run audit:skills
npm run audit:skill-security
npm run token:report
pre-commit run --all-files
```

Release/security checks:

```bash
semgrep scan --config p/default --metrics=off
osv-scanner scan source -r . --format json --output-file osv-results.json
gitleaks git --redact --report-format json --report-path gitleaks-report.json .
gitleaks dir --redact --report-format json --report-path gitleaks-dir-report.json .
npx knip
```

Installer checks should use sandbox paths only and must not write to real `$HOME` / `$USERPROFILE` in CI.
